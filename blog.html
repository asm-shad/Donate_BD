What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents an HTML document as a tree structure. It provides a way for JavaScript to interact with the elements, attributes, and content of a web page. Think of it as a hierarchical representation of the document, where the root is the element, and child elements are nested beneath it.

How do you select an element from the DOM?

There are several methods to select elements from the DOM:

getElementById(): Selects an element by its unique ID attribute.
getElementsByTagName(): Selects elements by their tag name.
getElementsByClassName(): Selects elements by their class name.
querySelector(): Selects the first element that matches a CSS selector.
querySelectorAll(): Selects all elements that match a CSS selector.

Example:

const paragraph = document.getElementById("myParagraph");
const links = document.getElementsByTagName("a");
const divs = document.getElementsByClassName("myDiv");
const firstDiv = document.querySelector(".myDiv");
const allDivs = document.querySelectorAll(".myDiv");

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a method where you add an event listener to a parent element, and that listener handles events for its child elements. Instead of adding separate listeners to each child, you rely on the parent to manage them all. This is especially useful when you have many child elements that need the same event to be handled (like clicks).

Why is it useful?
Better performance: You don’t need to add a listener to every child, which saves memory and makes your code run faster.
Handles dynamic content: It's easier to manage new elements added to the page (like list items). The parent’s listener will automatically handle these.
Cleaner code: Centralizing event handling on the parent makes the code simpler and easier to maintain.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element’s attributes and styles using the DOM through JavaScript.

Attributes:
getAttribute(): Gets the value of an attribute.
setAttribute(): Sets the value of an attribute.
removeAttribute(): Removes an attribute.

Example:
const image = document.getElementById("myImage");
console.log(image.getAttribute("src"));
image.setAttribute("alt", "A beautiful image");
image.removeAttribute("width");

Styles:
style property: Directly access and modify an element's inline styles.
classList property: Add, remove, or toggle classes on an element.
getComputedStyle(): Gets the computed style of an element, including inherited styles.

Example:
const paragraph = document.getElementById("myParagraph");
paragraph.style.color = "red";
paragraph.classList.add("bold");
paragraph.classList.remove("italic");
const computedStyle = window.getComputedStyle(paragraph);
console.log(computedStyle.fontSize);
